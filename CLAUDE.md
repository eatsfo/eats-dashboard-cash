## 🤖 데이터 블록 규칙 (2026-07-07 — 본사 대시보드 AI 연동, 자동 준수)
페이지(HTML)를 수정할 때마다, **그 페이지 본문 맨 아래에 아래 형식의 데이터 블록을 실제 값과 동기화**해서 유지하라(없으면 생성). 본사 대시보드 AI가 매주 이 블록을 읽어 경영 인사이트에 반영한다. 화면에는 보이지 않는다.

```html
<script type="application/json" id="fable-data">
{"asof":"YYYY-MM-DD","kpi":{"지표명(단위)":숫자},"risks":["위험 요약 한 줄"],"decisions":["결정 대기 사안 — 시한"]}
</script>
```
- `kpi` = 이 페이지의 핵심 수치 3~6개(페이지 표와 일치) · `risks` = 지금 위험한 것 · `decisions` = 경영진 결정이 필요한 것(시한 포함)
- JSON 유효성 유지(따옴표·쉼표). 페이지가 여러 개면 각 페이지마다 자기 블록.
